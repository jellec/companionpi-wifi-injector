Inject & eject

1. Copies wifi repo onto boot partition (no internet on RPi needed)
2. Injects firstrun.sh — installs from SD on first boot (~1 min)
3. Sets password on companion user (userconf.txt)
4. Companion already installed — starts automatically
5. WiFi AP active after reboot: connect to CompanionPi
⚠ Fetch the wifi repo first (Step 1) before injecting.

After injecting: eject the SD card safely, insert into RPi, and power on.
First boot takes ~1 minute — Companion starts automatically.
